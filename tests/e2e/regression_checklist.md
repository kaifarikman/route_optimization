# Чеклист граничных случаев тестирования UI

## 1. Блок генерации точек

### Количество точек
- 2 точки -> корректное построение маршрута
- 50 точек -> корректное построение маршрута
- 1 точка -> уведомление о недопустимом значении
- 51 точка -> уведомление о недопустимом значении
- Нецелое число (3.7) -> уведомление о недопустимом значении
- Отрицательное число (-5) -> уведомление о недопустимом значении
- Буквы / спецсимволы -> уведомление о недопустимом значении
- Пустое поле -> уведомление "введите количество точек"
- Ноль (0 точек) -> уведомление о недопустимом значении

### Радиус
- 5 км -> корректное построение маршрута
- 50 км -> корректное построение маршрута
- 0 км -> уведомление о недопустимом значении
- 10 000 км -> уведомление о недопустимом значении
- Отрицательный радиус -> уведомление о недопустимом значении
- Пустое поле -> уведомление "введите радиус"
- Дробные значения (0.5, 1.25) -> корректное построение маршрута

### Широта и долгота
- Широта = 57 -> корректное отображение на карте
- Широта = 50 -> корректное отображение на карте
- Долгота = 37 -> корректное отображение на карте
- Широта = 40 -> корректное отображение на карте
- Широта = 100 -> уведомление о недопустимом значении
- Широта = -100 -> уведомление о недопустимом значении
- Долгота = 200 -> уведомление о недопустимом значении
- Долгота = -200 -> уведомление о недопустимом значении
- Буквы / спецсимволы -> уведомление о недопустимом значении
- Пустое поле широты или долготы -> уведомление "введите широту/долготу"

## 2. Ручное добавление точек

- Нажатие на кнопку "Добавить точку" -> кликнув по карте, появляется новая точка
- Добавление точки в режиме, когда маршрут уже есть -> маршрут сбрасывается
- Добавление 51 точки вручную -> ошибка добавления точки
- Попытка добавить точку при 50 уже существующих точках -> ошибка добавления точки

## 3. Импорт точек из файла

- Загрузка JSON-файла с точками -> точки отображаются на карте 
- Загрузка CSV-файла с точками -> точки отображаются на карте
- Загрузка пустого файла/ файла с 1 точкой -> уведомление об ошибке
- Возможность загрузить файл другого расширения -> возможно загрузить только JSON/CSV
- Файл с 51 точкой -> загружаются первые 50 точек из файла
- Файл с точками, содержащими некорректные координаты -> точки не отображаются на карте
- Импорт поверх существующих точек -> на карте остаются только точки из файла

## 4. Импорт точек
- Импорт JSON-файла -> файл успешно импортируется, точки в нем корректны 
- Импорт CSV-файл -> файл успешно импортируется, точки в нем корректны 
- Скопировать ссылку -> ссылка копируется, при её открытии отображается маршрут на карте

## 5. Сброс, очистка, повторные действия

- Обновление страницы -> точки исчезают 
- Кнопка "Новый маршрут" -> удаляет точки и маршрут
- Ошибка генерации -> поля остаются заполненными для повторной попытки

## 6. Масштабируемость

- Очень маленькое окно браузера -> карта не ломается, кнопки доступны
- Экран с высоким зумом -> элементы не наезжают друг на друга
