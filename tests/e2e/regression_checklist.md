# Чеклист ручной проверки UI

Список граничных случаев, которые стоит прогнать руками перед релизом. Стрелка `->` показывает ожидаемое поведение.

## 1. Генерация точек

### Количество точек
- 2 точки -> маршрут строится
- 50 точек -> маршрут строится
- 1 точка -> уведомление о недопустимом значении
- 51 точка -> уведомление о недопустимом значении
- нецелое число (3.7) -> уведомление о недопустимом значении
- отрицательное число (−5) -> уведомление о недопустимом значении
- буквы и спецсимволы -> уведомление о недопустимом значении
- пустое поле -> уведомление «введите количество точек»
- ноль -> уведомление о недопустимом значении

### Радиус
- 5 км -> маршрут строится
- 50 км -> маршрут строится
- 0 км -> уведомление о недопустимом значении
- 10 000 км -> уведомление о недопустимом значении
- отрицательный радиус -> уведомление о недопустимом значении
- пустое поле -> уведомление «введите радиус»
- дробные значения (0.5, 1.25) -> маршрут строится

### Широта и долгота
- широта 40 / 50 / 57 -> точки корректно ложатся на карту
- долгота 37 -> корректно на карте
- широта 100 или −100 -> уведомление о недопустимом значении
- долгота 200 или −200 -> уведомление о недопустимом значении
- буквы и спецсимволы -> уведомление о недопустимом значении
- пустая широта или долгота -> уведомление «введите широту/долготу»

## 2. Ручное добавление точек

- кнопка «Добавить точку», затем клик по карте -> появляется новая точка
- добавление точки, когда маршрут уже построен -> маршрут сбрасывается
- попытка добавить 51-ю точку -> ошибка добавления

## 3. Импорт точек из файла

- JSON с точками -> точки появляются на карте
- CSV с точками -> точки появляются на карте
- пустой файл или файл с одной точкой -> уведомление об ошибке
- файл другого расширения -> загрузить можно только JSON/CSV
- файл с 51 точкой -> загружаются первые 50
- файл с битыми координатами -> такие точки не появляются на карте
- импорт поверх существующих точек -> на карте остаются только точки из файла

## 4. Экспорт и публичная ссылка

- экспорт в JSON -> файл скачивается, содержимое корректно
- экспорт в CSV -> файл скачивается, содержимое корректно
- «Поделиться» -> ссылка копируется; при открытии она показывает маршрут на карте

## 5. Сброс и повторные действия

- обновление страницы -> точки исчезают
- кнопка «Новый маршрут» -> удаляет точки и маршрут
- ошибка генерации -> поля остаются заполненными для повторной попытки

## 6. Адаптивность

- очень маленькое окно браузера -> карта не ломается, кнопки доступны
- высокий зум экрана -> элементы не наезжают друг на друга
